# Swiss Job Market BI Dashboard

A BI project that pulls Swiss job postings from the Adzuna API (and jobs.ch
where feasible), cleans and structures the data with pandas, loads it into
SQLite, and visualizes in Power BI which skills, roles, and regions are most
in demand — split by Working Student vs. Full-Time postings.
